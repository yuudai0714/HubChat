ヘルプ・Q&A

サービスの追加

左下の ＋ ボタンをクリックし、追加したいサービスを選択します。カテゴリタブで絞り込みも可能です。

サービスの切り替え

左サイドバーのアイコンをクリックすると、そのサービスに切り替わります。

サービスの並び替え

サイドバーのアイコンをドラッグ＆ドロップで好きな順番に並び替えられます。

サービスの削除・非表示

サイドバーのアイコンを右クリックすると、メニューが表示されます。「非表示」でサイドバーから隠し、「削除」で完全に除去できます。

ショートカットキー

Cmd+Shift+↓（Mac）/ Ctrl+Shift+↓（Windows）で次のサービスに切り替えます。

Cmd+Shift+↑（Mac）/ Ctrl+Shift+↑（Windows）で前のサービスに切り替えます。

ズーム（表示サイズ調整）

Cmd + -（Mac）/ Ctrl + -（Windows）で表示を縮小します。

Cmd + 0（Mac）/ Ctrl + 0（Windows）で表示サイズをリセットします。

※ズームレベルはサービスごとに個別に保存されます。

戻る・進む・リロード

画面の右上にマウスを移動すると、ナビゲーションバーが表示されます。

◀ 戻る　▶ 進む　↻ リロード

マウスを離すと自動的に非表示になります。

Outlook・MS Teamsのログイン

HubChat内ではMicrosoftの認証方法に制限があります。

✅ 利用できる方法

・メールにコードを送信する

❌ 利用できない方法

・顔認証 / 指紋認証 / PIN / セキュリティキー
・パスキー（パスワードレス認証）
・パスワード入力
・モバイルアプリでの承認

ログイン手順

1. メールアドレスを入力
2.「別の方法でサインインする」を選択
3.「○○@gmail.com にコードを送信する」を選択
4. 届いたコードを入力

Slackワークスペースの追加

Slackを追加すると、ワークスペースURL入力ダイアログが表示されます。

ワークスペースURLの形式: your-workspace.slack.com

URLがわからない場合は、ブラウザでSlackにログインし、アドレスバーのURLを確認してください。

テーマの切り替え

左下の ⚙（設定）ボタンをクリックし、設定画面の「テーマ」セクションでダーク / ライトを切り替えられます。

画面が白い・読み込まれない

右上のナビゲーションバーからリロード（↻）を試してください。改善しない場合は、サービスを一度削除して再追加してください。

ログインできない

一部のサービスはCookie/セッションの問題でログインが切れることがあります。リロードするか、サービスを再追加してください。

Canva で Google ログインができない

Googleのセキュリティにより、デスクトップアプリ内でのGoogleログインがブロックされる場合があります。Canvaのログイン画面で「別の方法でログイン」→「メールアドレスでログイン」を選択してください。Googleアカウントで作成したCanvaアカウントでも、Canvaのパスワードリセットページからパスワードを設定できます。

各サービスで Google ログインができない場合

Googleのセキュリティポリシーにより、デスクトップアプリ内からのGoogleログイン（OAuth認証）がブロックされる場合があります。これはHubChatに限らず、同種のデスクトップアプリ共通の制限です。各サービスのメールアドレス＋パスワード方式でログインしてください。Googleアカウントで登録したサービスでも、各サービスの「パスワードをお忘れの場合」からパスワードを設定できます。

通知が来ない

各サービス側の通知設定が有効になっているか確認してください。HubChatはサイドバーのアイコンにバッジ（赤い数字）で未読数を表示します。

アプリが固まった

アプリを一度終了し、再起動してください。
